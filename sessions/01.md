### Du code dans un fichier

Oups, les variables disparaissent quand on ferme le REPL.
Solution: les mettre dans un fichier `.py`.

```python
a = 1
b = 2

c = a+b

print(c)
```

S'assurer que les étudiants peuvent:

* `cd` dans le bon répertoire
* Lancer `python <lefichier.py`.

Notions:

* Fonction print()
* Commentaires

* Définir notre propre fonction:

    * Paramètres
    * Indentation
    * Deux-points
    * return()

```python
# ceci est un commentaire
def add(a, b):
  return a + b

a = 1
b = 2
c = add(a, b)
print(c)
```

### Entrèe standard

* input()

Démo: "à quel nombre je pense"

```python
secret = 42

print("Devine le nombre auquel je pense"):
while True:
    reponse = input()
    if response > secret:
         print("Trop grand")
    if response < secret:
        print("Trop petit")
    print("Gagné")
    break
```
